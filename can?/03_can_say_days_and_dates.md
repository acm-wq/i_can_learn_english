## I can say days and dates?

| **Days of the week**        | **Months of the year**                                           | **Seasons**                  |
|-----------------------------|------------------------------------------------------------------|------------------------------|
| 🗓️ **Monday**               | 📅 **January** **February** **March** **April** **May**          | 🌿 **Spring**                |
| 🗓️ **Tuesday**              | 📅 **June** **July** **August** **September**                    | 🍂 **Autumn**                |
| 🗓️ **Wednesday**            | 📅 **October** **November** **December**                         | ❄️ **Winter**                |
| 🗓️ **Thursday**             |                                                                  | 🌸 **Summer**                |
| 🗓️ **Friday**               |                                                                  |                              |
| 🗓️ **Saturday**             |                                                                  |                              |
| 🗓️ **Sunday**               |                                                                  |                              |

-- --

- Numbers 1, 21, 31 get the ending -st. 
- Numbers 2, 22 get the ending -nd.
- Numbers 3, 23 get the ending -rd. 
- All others get the ending -th. Only 12 — twelfth, not twelveth.


```
We write dates like this:
10 March || 10th March || 3.10.25 || 3/10/25
We say the date like this:
What's the day today? ~ It's March the tenth
Say the year like this:
1980 – Nineteen eighty
1999 – Nineteen ninety-nine
2000 – Two thousand
2013 – Two thousand and thirteen
```
