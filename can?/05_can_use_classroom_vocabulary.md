### ❓ Basic Questions for Lesson

#### Asking for Permission

- Can I go to the bathroom, please? 🚻
- Can I open/close the window? 🪟❄️☀️
- Can I drink some water? 💧
- Can I leave when the class is over? ⏳
- Can I borrow a pen/pencil, please? ✏️🖊️
- May I sit here? (More polite) 😊

#### Understanding & Asking for Help

- Sorry, I don’t understand. Can you repeat, please? 🤔🔄
- How do you say this word? 📖🗣️
- What does this word mean? 🤷‍♂️
- Can you spell it, please? 🔤
- How do you pronounce this word? 🔊
- Can you write it on the board? 🏫🖊️

#### Asking About the Lesson

- What page are we on? 📖📄
- Can we listen to the audio again? 🎧🔄
- Is this correct? ✅❌
- Do I need to write this down? ✍️📒
- Can you give another example? 📝
- What’s the homework for today? 📚📝
- When is the next test? 😱📆

#### Classroom Interactions

- Can I work with a partner? 👯‍♂️
- Can we do this exercise together? 🤝
- Excuse me, I have a question! ✋🤔
- Can I answer this question? 🙋‍♀️
- Sorry, I’m late! ⏰😓
